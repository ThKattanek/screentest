Welcome to the Screentest Home Page

What is Screentest?

Screentest is a CRT and LCD screen testing utility. Ever wondered how good the quality (sharpness, linearity, convergence, etc.) of your CRT/LCD monitor is? Screentest displays the testing patterns which you can use to evaluate the quality of your monitor.

How can I get Screentest?

Use the SourceForge Downloads page.

Development repository

The screentest sources are maintained in a git repository. You can get the sources with git using this command:

git clone git://screentest.git.sourceforge.net/gitroot/screentest/screentest

You can also browse the repository with gitweb.
